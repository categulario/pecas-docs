Herramientas

Digitalización

  • pc-tiff2pdf: utiliza el poder de Libtiff y de Tesseract para crear archivos PDF con OCR o TXT a partir de imágenes TIFF.

Archivos madre

  • pc-add: añade plantillas CSS o archivos JavaScript al proyecto.

  • pc-analytics: analiza archivos EPUB, XML, XHTML o HTML para un mayor cuidado editorial y técnico.

  • pc-images: redimensiona o comprime distintos tipos de imágenes.

  • pc-pandog: convierte entre archivos MD, HTML, HTM, XHTML, XML, JSON, TeX, EPUB, ODT o DOCX, estos tres últimos tipos a través de Pandoc.

EPUB

  • pc-automata: automatiza el flujo de trabajo al poder usar todos los scripts para EPUB con una sola línea de comandos.

  • pc-creator: crea un proyecto para EPUB con distintas opciones.

  • pc-divider: separa un documento HTML cada <h1> o <section>.

  • pc-notes: agrega de manera automatizada las notas al pie a archivos con sintaxis tipo HTML o documentos TeX.

  • pc-cites: está en desarrollo…

  • pc-index: agrega índices índices analíticos a archivos HTML, HTM, XHTML, XML o TeX.

  • pc-recreator: recrea los archivos OPF, NCX y NAV así como crea o recrea el archivo EPUB.

  • pc-changer: cambia versiones de EPUB entre 2.0.0, 2.0.1, 3.0.0 y 3.0.1.

Estado de Pecas y sus dependencias

  • pc-doctor: analiza el estado de Pecas y sus dependencias.
